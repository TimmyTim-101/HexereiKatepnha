## 使用说明
1. 前往[Release](https://github.com/TimmyTim-101/HexereiKatepnha/releases)页面下载最新程序压缩包
2. 解压缩到任意文件夹
3. 点击可执行程序**HexereiKatepnha.exe**运行程序

> [!IMPORTANT]
> Release本身不包含用户数据<br>
> 请通过手动移动旧版本程序所在文件夹中**Configs**文件夹到新版本程序所在文件夹中来迁移用户数据
